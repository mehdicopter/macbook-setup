# Mac OS X Setup

This is what I install on my MacBookPro.

## Prerequisites
```sh
mkdir -p ~/{Github,Bitbucket,workspace}
```

## Homebrew
```sh
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

## Tools
```sh
brew install git
brew install gnu-sed
brew install vim
brew install screen
brew install p7zip
brew install tree
```
